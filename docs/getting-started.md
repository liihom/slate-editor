# 快速开始

## CSlate

CSlate 是一个基于[Slate](https://github.com/ianstormtaylor/slate) 构建的富文本编辑器。你可以通过插件的方式轻松自定义节点、工具等元素。

## 安装

```shell
$ npm i @cslate/cslate --save # Ant 风格
```
